О сервисе

Данный веб-сервис разработан в рамках преддипломной практики по теме «Разработка системы для автоматизированного формирования технического задания в соответствии с ГОСТ 19.201-78 (на примере Частного образовательного учреждения высшего образования „Московский университет имени С. Ю. Витте“)».

Цель сервиса — помочь разработчику и заказчику информационной системы формировать текстовые разделы технического задания (ТЗ) в едином стиле, с опорой на требования ГОСТ 19.201-78, типовые примеры и методические материалы университета. Система позволяет на основе краткого описания проекта и выбора целевого раздела ТЗ получить согласованный по структуре и терминологии текст, который затем может быть уточнён и включён в итоговый документ.

Используемые технологии

  • Flask — лёгкий веб-фреймворк на языке Python для построения сервиса.
  • Механизм Retrieval-Augmented Generation (RAG) — поиск релевантных фрагментов ГОСТ и методических материалов с последующей генерацией текста на их основе.
  • TF–IDF-индекс и косинусное сходство — для отбора фрагментов корпуса, близких к описанию конкретного проекта.
  • Proxy API ChatGPT — для генерации текстов разделов ТЗ с учётом структурных требований стандарта.

Структура работы сервиса

  1. Пользователь задаёт параметры проекта: название, предметную область, краткое описание.
  2. Выбирается раздел технического задания (например, «Основания для разработки», «Назначение системы», «Требования к системе»).
  3. Сервис, в зависимости от выбранного режима (без RAG, RAG только по ГОСТ или RAG по всем источникам), подбирает фрагменты корпуса и формирует запрос к модели.
  4. Модель возвращает связный текст раздела, который отображается в интерфейсе и может быть скопирован в документ ТЗ.

Автор и назначение

Сервис предназначен для демонстрации навыков проектирования и реализации интеллектуальных информационных систем, а также для практического применения при подготовке технических заданий на разработку программного обеспечения для нужд университета. ФИО студента и научного руководителя указываются в нижнем колонтитуле (подвале сайта).
